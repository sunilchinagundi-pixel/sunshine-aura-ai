# Sunshine Aura AI — Training & Consultancy Platform

A modern platform for AI training, consultancy services, job placement, and learning.

**Tech Stack:**
- Backend: Python + FastAPI
- Frontend: React + Vite
- API: RESTful endpoints

## Quick Start

### Backend Setup

```bash
cd backend
.\.venv\Scripts\activate
pip install -r requirements.txt
python main.py
```

Visit: http://127.0.0.1:8000/docs (API docs)

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Visit: http://localhost:5173

## Features

- User registration and login (free & pro tiers)
- Free and paid training courses
- Job board
- Consulting services
- Pricing tiers
- API-first design for future Android app
